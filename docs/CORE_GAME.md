# Dominia Simulator - Core Game

## Filosofia do Projeto
Dominia Simulator é um RPG Auto Battler Roguelite inspirado em Daggerheart.

O jogador toma decisões estratégicas fora do combate.
Os combates são simulados automaticamente.

## Pilares
- Construção de Build
- Adaptação ao RNG
- Progressão através de escolhas

## Atributos

### Primários
- Força
- Conhecimento
- Agilidade

### Defensivos
- Armadura
- Defesa Mágica

### Derivados
- HP
- Evasão
- Iniciativa

Evasão = 10 + Agilidade + Modificadores

## Slots
- 3 Poderes Equipados
- 3 Itens Equipados
- Inventário

## Progressão
Após a maioria das batalhas:
- Escolher Item
OU
- Escolher Poder
